# Machine Learning in Education - Final Project

This repository contains the code and data for the final project of the course
"Machine Learning in Education".

## Project Topic
Educational Data Problem Definition and Model Analysis

## Dataset
The dataset used in this project is the Student Performance Dataset from the
UCI Machine Learning Repository.

- Source: UCI Machine Learning Repository
- Data type: Educational data collected from secondary school students
- Size: 395 students with 33 attributes
- Data includes student background, learning behavior, and academic performance

## Files
- data/student-mat.csv  
  Student performance dataset used for analysis

- data/test_load.py  
  Python script for loading the dataset and basic data inspection

## Description
This project focuses on defining educational data analysis problems and applying
basic supervised and unsupervised learning methods to explore student learning
patterns and academic performance.

The main purpose of this repository is to support the presentation and report
submitted for the final course project.

## Notes
This repository is created for academic coursework purposes.
